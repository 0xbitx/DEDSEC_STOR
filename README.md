<h1 align="center"> DEDSEC_STOR</h1>
<h4 align="center">STOR - Dark Web Hosting Toolkit</h4>

## DESCRIPTION
STOR is a Linux-based command-line toolkit engineered for seamless deployment of websites as Tor hidden services on the dark web. Designed specifically for security researchers, penetration testers, privacy advocates, digital activists, journalists, and individuals seeking to host content anonymously on the Tor network while maintaining robust operational security standards.

## Features
  Core Functionality

  * One-Click Deployment - Deploy websites to Tor network with minimal configuration
  * Automatic .onion Address Generation - Get your unique Tor address automatically
  * Built-in Web Server - Integrated HTTP server for serving content
  * Site Management - Create, update, and redeploy existing sites
  * History Tracking - Keep records of all deployments

  Security & Privacy

  * Automatic Tor Installation - Installs and configures Tor service automatically
  * Secure File Permissions - Sets proper permissions for hidden service directories
  * Backup System - Automatic backups of deployed sites
    
  Management Tools

  * Interactive Menu System - User-friendly terminal interface
  * Site Information Display - Detailed stats about deployed websites
  * Progress Visualization - Real-time progress bars for deployments
  * Color-Coded Output - Easy-to-read terminal output with colors
  * Deployment History - View and manage past deployments

## INSTALLATION 
    * git clone https://github.com/0xbitx/DEDSEC_STOR.git
    * cd DEDSEC_STOR
    * sudo pip3 install tabulate
    * chmod +x dedsec_stor
    * sudo ./dedsec_stor
